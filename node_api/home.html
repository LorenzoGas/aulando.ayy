Benvenuto nelle API di aulando.ayy!

Grazie a queste API gratuite potrai avere a disposizione informazioni riguardo gli orari delle varie aule nei vari dipartmenti dell'universit√† di trento.

Ecco la lista dei diversi servizi:

  • /auleLibere -> lista delle aule libere in un dato dipartimento, un determinato giorno e alla determinata ora. Parametri:
    • 'formato': formato dell'output, JSON o XML; default JSON
    • 'dipartimento': id del dipartimento desiderato
    • 'giorno': giorno interessato nel formato 'aaaa-mm-gg'
    • 'ora': ora interessata nel formato 'hh:mm'
  • /auleLibereDalleAlle -> Restituisce una lista di aule disponibili nell'intervallo di tempo specificato. Parametri:
    • 'formato': formato dell'output, JSON o XML; default JSON
    • 'dipartimento': id del dipartimento desiderato
    • 'giorno': giorno interessato nel formato 'aaaa-mm-gg'
    • 'dalle': ora iniziale dell'intervallo desiderato nel formato 'hh:mm'
    • 'alle': ora iniziale dell'intervallo desiderato nel formato 'hh:mm'
  • /orariAula -> Restituisce una lista di lezioni per un'aula specificata in un giorno specificato. Parametri:
    • 'formato': formato dell'output, JSON o XML; default JSON
    • 'dipartimento': id del dipartimento desiderato
    • 'giorno': giorno interessato nel formato 'aaaa-mm-gg'
    • 'aula': nome dell'aula interessata
  • /listaDipartimenti -> Restituisce una lista di dipartimenti. Parametri:
    • 'formato': formato dell'output, JSON o XML; default JSON
  • /listaAule -> Restituisce una lista di aule del dipartimento specificato. Parametri:
    • 'formato': formato dell'output, JSON o XML; default JSON
    • 'dipartimento': id del dipartimento desiderato
  • /listaDocenti -> Restituisce una lista di docenti per il dipartimento specificato. Parametri:
    • 'formato': formato dell'output, JSON o XML; default JSON
    • 'dipartimento': id del dipartimento desiderato
  • /listaCorsi -> Restituisce una lista di corsi per il dipartimento specificato. Parametri:
    • 'formato': formato dell'output, JSON o XML; default JSON
    • 'dipartimento': id del dipartimento desiderato
  • /listaSubcorsi -> Restituisce una lista di subcorsi per il dipartimento specificato. Parametri:
    • 'formato': formato dell'output, JSON o XML; default JSON
    • 'dipartimento': id del dipartimento desiderato
